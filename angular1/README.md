ustadium-webapp
=====================

Ustadium.com app

in order to run the app use the package.json scripts:
to build dev env:

`npm run build:dev`
and then run the command :
`npm run dev:server`

for production:

`npm run build:prod`

and then

`npm run prod:server`

the live app is `usatdium.com`

deployed on heroku
